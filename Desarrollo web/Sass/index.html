Comandos para sass:

  • Descargas desde Extensiones sass
  • Ejecutar el siguiente comando: sass --watch styles.scss styles.css
  • Te debe crear el archivo styles.css
  • Si da error, ejecutar en PowerShell como administrador el siguiente comando: Set-ExecutionPolicy RemoteSigned

Cada que se hace un cambio se debe ejecutar el comando "sass --watch styles.scss styles.css"

Esto ayudara a que los cambios se visualicen en el styles.css
